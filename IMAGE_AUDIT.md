## 3. SITEWIDE VISUAL DIRECTION

### Brand Image Standards (Based on MAXHUB-Style Benchmark)

#### Composition Rules
- **No humans** in any imagery (product-first focus)
- **Centered displays** - screens should be the hero element
- **Wide-angle room shots** showing complete AV ecosystem
- **Clean sightlines** - no clutter, cables hidden
- **Consistent aspect ratio** - 16:9 for all major images

#### Lighting Standards
- **Natural daylight** preferred with soft shadows
- **Even illumination** - no harsh shadows on screens
- **Screen visibility** - displays should be clearly visible, not blown out
- **Cool color temperature** (5000-6500K) matching corporate aesthetic

#### Room Style Guidelines

| Room Type | Key Visual Elements | Display Size |
|-----------|-------------------|--------------|
| Huddle | Small table, 2-4 chairs, wall-mounted display | 55-65" |
| Meeting | Conference table, 6-8 chairs, prominent display | 65-75" |
| Boardroom | Executive table, premium finishes, large display | 86-98" |
| Classroom | Rows of desks, teaching display at front | 75-86" interactive |
| Training | Flexible seating, mobile displays | 65-75" on stand |
| Auditorium | Tiered seating, large projection/display | 100+" or projection |

#### Color Palette Alignment
- **Primary:** Blue tones (#2563eb, blue-600)
- **Backgrounds:** Cool grays, white walls
- **Accents:** Natural wood or black furniture
- **Screens:** Showing UI mockups with blue gradients

#### Technical Requirements
- **Resolution:** Minimum 1920x1080, preferred 2400x1600
- **Format:** JPG for photos, PNG for diagrams
- **Optimization:** 80-85% quality for web
- **Alt text:** Descriptive, SEO-optimized
