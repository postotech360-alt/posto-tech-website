## 🎨 PLATFORM-SPECIFIC TIPS

### Midjourney v6
- Use `--ar 16:9` for all images
- Add `--style raw` for photorealistic results
- Use `--q 2` for maximum quality
- Use `--v 6` for latest version
- Add `--no people, humans, persons` to ensure no humans

### DALL-E 3
- Be very specific about "no people"
- Mention aspect ratio explicitly
- Describe lighting in detail
- Use professional photography terms

### Adobe Firefly
- Use "Photo" style setting
- Set aspect ratio to 16:9
- Use "Professional" visual intensity
- Add "commercial photography" to prompts

### Stable Diffusion XL
- Use photorealistic checkpoint (e.g., RealVisXL)
- CFG Scale: 7-8
- Steps: 30-50
- Negative prompt: "people, humans, persons, blurry, low quality"
